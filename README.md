# Python Lead & Social Media Extractor

A powerful automation tool designed to extract contact information (Emails) and Social Media profiles (LinkedIn, Facebook, Twitter, Instagram) from any given website URL.

## üöÄ Key Features
- **Email Extraction**: Uses advanced Regex patterns to find emails in text and mailto links.
- **Social Media Detection**: Automatically identifies links to major platforms.
- **Data Normalization**: Handles varied URL formats (with or without https).
- **CSV Export**: Saves all found leads into a structured `leads.csv` file.
- **Excel Export**: Saves all found leads into a structured `leads.xlsx` file.

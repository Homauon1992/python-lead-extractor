# python-lead-extractor
Automated tool to extract emails and social media leads from websites using Python.

A powerful automation tool designed to extract contact information (Emails) and Social Media profiles (LinkedIn, Facebook, Twitter, Instagram) from any given website URL.

- **CSV Export**: Saves all found leads into a structured `leads.csv` file.
